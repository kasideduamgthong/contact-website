# Contact Website

Professional portfolio website built using Flask and Bootstrap.

## Features
- Flask Framework
- Bootstrap 5
- SQLite Database
- Contact Form (Data saved to database)
- Blog System
- Testimonials
- Admin Dashboard

2. About

7. Messages
8. Blog

10. Dashboard
